Wasting Time with CSS
February 8th, 2016

For at least half a year, I have used complex CSS to limit the number of words per line, a feature which most webpages have. I would use a @media query, such as the following:

@media screen and (min-width: 700px) {
  #inner {
    width: 700px;
    margin: auto;
  }
}

This method takes up 6 lines of code, and complexifies the whole CSS file. However, recently I found out that using the max-width property, I could compress this code quite a bit:

#inner {
  max-width: 700px;
  margin: auto;
}

This does effectively the same thing. Using this formula not only compresses the code, but if I were to change the max width, I would have to change only one property, as opposed to the previous method.

I wonder...how much time would I have saved to have known that in the first place?
